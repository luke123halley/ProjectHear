What is Project Hear

A small device which would sit in a music studio alongside a producer. The function of the device would be to prevent hearing damage. The device would warn the producer if the DB level currently being outputted was at an unsafe level which could cause permanent damage to his or her's ears.

Tools, Technologies and Equipment

  • Intel Galileo - Device to run the program on
  • Python - to write the program
  • Microphone Module - To detect the DB level
  • LED's - To Indicate if the level is at a safe or unsafe level
